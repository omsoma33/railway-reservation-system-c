# 🚆 Railway Reservation System (C Project)

## 📌 Description
This project is a menu-driven Railway Reservation System implemented in C using:
- Linked List
- Stack
- Circular Queue

## 🧠 Features
- Ticket Booking
- Ticket Cancellation
- Waiting List Management
- Display Passenger Records

## ⚙️ Data Structures Used
- Linked List → Stores booked passengers
- Stack → Stores cancelled tickets
- Circular Queue → Handles waiting list

## ▶️ How to Run
1. Open the project in Turbo C / CodeBlocks
2. Compile the file railway.c
3. Run the program
4. Use menu options
